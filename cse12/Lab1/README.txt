Lab1: Intro to Logic Simulation

DESCRIPTION
Introduction to Multimedia Logic (MML) by 
	implementing logic (POS/SOP, NAND and 
	NOR implementation) based on truth tables
	using senders and receivers to minimize
	clutter.

FILES
- Lab1.lgi: This file includes the logic
	circuits for the lab.
	
INSTRUCTIONS
Open Lab1.lgi with MML and navigate to the 
	first page. Click the green "Run simulation" 
	button found in the toolbar.
