Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab, the user can save a value into one of four 4-bit registers. This value is either a keypad user input, or the output of the ALU. The ALU is a 4-bit bitwise right rotation that takes two of the register values as inputs.

-----------
FILES

-
Lab2.lgi

This file includes the circuitry for the lab.

-----------
INSTRUCTIONS

The program is intended to be run by MultiMedia Logic (MML). Open Lab2.lgi with MML and the first page should appear. Run the simulation and input values into the registers using the keypad, switches, and update and clear buttons.
